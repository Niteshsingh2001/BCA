Computer For Everyone

Computer For Everyone

Pre-Requisite → Knowledge of Computers

Subject Area → Computer Application

Objective → To provide Computer skills and knowledge for all the students and to enhance and understand the usefulness of essential office tools.

Course Outcome

  • A student who successfully fulfills the course requirements will be able to:
    1. Understand the fundamental concept of information technology and working on computers to learn about components and internal working of computers.
    2. Perform documentation crafting professional word documnets, Resizing images and formatting it.
    3. Perform accounting operations and Performance Analysis.
    4. Create seminar and Business Presentations.
    5. Gather data using Google forms and generating reports.

Details of the course ↓

Unit No.

CONTENT

1

Basic Computer Concepts

  • Computer Definition
  • Its characteristics
  • Input/Output
  • Storage units
  • Devices ↓
    • Input Devices → Keyboard, Mouse, Joystick, Scanner, Web cam.
    • Output Devices → Monitors, Printers, laser, Ink-jet, Joystick.

2

Introduction to Ms Word and its new versions

  • Navigating Word upper versions, opening new documents and existing documents, templates, Basic and advanced text, paragraph, and document formatting, insert images, graphics, and video, MS Word as a photo editor, creating impressive tables using new formatting tools.

3

Introduction to Ms Excel

  • Creating spreadsheets, Format cells, rows, columns, and entire worksheets, Enter data into a spreadsheet, Use formulas and functions for math, accounting, and totaling, Create formulas and functions, Create charts and diagrams for data, Create charts and diagrams for your data.

4

Introduction to Ms PowerPoint

  • Navigate the PowerPoint interface, Create new presentations from scratch ,add text, pictures, sounds, movies, and charts to your presentations, Design slides using themes, colors, and special effects, add special effects to slide transitions, Concept of Master Slide.

5

Internet Usage

  • WWW and Web Browsers: Web Browsing software, Surfing the Internet, Chatting on Internet, Basic of electronic mail, Using Emails, Document handling, Network definition, Common terminologies: LAN, WAN, Node, Host, Network Components: Severs, Clients, Communication Media.
