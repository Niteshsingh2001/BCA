Basic Computer Concepts

Computer

  • Definition → A computer is a programmable machine designed to perform arithmetic and logical operation automatically and sequentially on the input given by the user and gives the desired output after processing.

Characteristics of computer

  • Accuracy ⇒ Computer always give accurate result depend upon data provided.
  • Speed ⇒ It can perform millions of instructions/operation per second, which takes a lot of time if done manually.
    • Speed of computer is given in nano secconds
      1 nano second = 1 x 10-9sec

Input/Output

  • Input → Data which goes in CPU for processing.
  • Output → Data which comes out of CPU.
