Class

  • Class is a way to bind the data and its associated function together.
  • It allow data and functions to be hidden if necessary from external use.
  • Generally, class specification has two parts:
    1. Class declaration
    2. Class function definition
      1. Inside the class definition
      2. Outside the class definition

Class declaration syntax

                       
                       
                        class class_name
                        {
                            private:
                                data_type1 variable_name;
                                data_type2 variable_name;
                            public:
                                data_type1 variable_name;
                                data_type2 variable_name;
                        };

Example ↓

                       
                       
                        class item
                        {
                            int num; // these will be private by default, as we have not specified.
                            float cost;
                            public:
                                void getdata(int a, int b);
                                void putdata(int);
                        };
